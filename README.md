DevCampus – Smart College Website (React + TypeScript + AI Integration)

DevCampus is a modern, full-stack college website built using React.js and TypeScript, designed to enhance the digital experience for students, parents, and administrators. It provides seamless access to college information, admissions, and inquiry handling through advanced AI-powered features.

The platform integrates both Vapi Voice AI calling and a Chat AI service, enabling users to interact via voice or text for instant admission inquiries, course details, and support assistance.

The backend is powered by MongoDB Atlas, ensuring secure, scalable, and efficient data storage for user inquiries, applications, and communication history.

🚀 Key Features
Built with React + TypeScript for a fast and scalable frontend
Responsive and modern UI for all devices
Chat AI assistant for instant text-based queries
Vapi Voice AI calling integration for voice-based inquiries
Inquiry management system for admissions and support
Secure backend using MongoDB Atlas
Real-time communication support for better user engagement
🛠️ Tech Stack
Frontend: React.js, TypeScript, JavaScript
Backend: Node.js, Express.js
Database: MongoDB Atlas
AI Services: Chat AI (LLM-based assistant), Vapi Voice AI
🎯 Purpose

To create an intelligent, interactive college platform that simplifies communication, automates inquiry handling, and provides students with both chat and voice AI support for a smarter admission experience.
